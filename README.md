# Orbit
Orbit is a small Unity project made to demonstrate how applying gravitational forces to an object orbiting around an immovable moon in space naturally puts it into a Keplerian orbit. It also was used in conjunction with a slideshow (unprovided in this repo) to highlight the importance of an accurate integration system such as the used Verlet integration instead of Euler.
